Sing Screen Layout Reference (Modern Theme)

This page documents the current sing layout model in UPlayerLayout.pas and UThemes.pas, using constants from [SingPlayerGrid] and [SingPlayerWidgetPlacement].

Core Formulas

grid = GetSingPlayerGrid(players, areaW, areaH, reserveTopLyricsSpace)
slotHeight = (areaH - rowGap*(rows-1)) / rows
maxLineSpacing = slotHeight / LaneGuideLineCount
slotScale = min(1, min(columnWidth/WidgetScaleReferenceW, maxLineSpacing/LaneBaseLineSpacing))
contentScale = slotScale * max(WidgetScaleMin, 1 - (players-1)*WidgetScalePerPlayer)
noteLineSpacing = min(maxLineSpacing, LaneBaseLineSpacing*contentScale)
guideTopY = slotTop + (slotHeight - LaneGuideLineCount*noteLineSpacing)/2
rowAnchorY = guideTopY + LaneRowAnchorGuideIndex*noteLineSpacing
